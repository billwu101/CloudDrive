# Frontend Search 模組任務

## 完成定義

- 使用者可從 TopSearchBar 搜尋項目。
- 搜尋具有 debounce、篩選、分頁與狀態顯示。
- 搜尋結果可開啟資料夾或預覽檔案。

## 最小可執行任務

- [ ] 建立 SearchFilters type。
- [ ] 建立 SearchResultPage 或搜尋結果區。
- [ ] 建立 `useSearchItems` query。
- [ ] 建立 debounce hook。
- [ ] 將搜尋字串同步到 URL query。
- [ ] 清空搜尋字串時停止 query。
- [ ] 建立 item type 篩選。
- [ ] 建立 MIME type 篩選。
- [ ] 建立搜尋結果列表。
- [ ] 建立搜尋分頁控制。
- [ ] 點擊資料夾時進入資料夾。
- [ ] 點擊檔案時開啟 preview。
- [ ] 建立 loading state。
- [ ] 建立無結果 state。
- [ ] 建立 error state。
- [ ] 加入 retry。

## 測試任務

- [ ] 測試 debounce。
- [ ] 測試空字串不查詢。
- [ ] 測試 type 篩選。
- [ ] 測試 MIME type 篩選。
- [ ] 測試搜尋結果開啟資料夾。
- [ ] 測試搜尋結果開啟預覽。
- [ ] 測試無結果狀態。
